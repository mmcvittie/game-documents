# Game Documentation

System reference docs for multiple game lines built on a shared EZD6-style core.

Run `npm run generate:docs` to regenerate all docs.

## How To Use This Folder

- Start with a system file below based on tone and mechanics.
- In each file, jump to `Core Rules` first, then `Character Creation`, then `Abilities`.
- Use `Gear & Equipment` and `Reference Tables` during play.
